Prueba de verificación — Admitad

Esta página contiene la meta-tag de verificación que pide Admitad:

<meta name="verify-admitad" content="e3b1ce5433" />

Si Admitad detecta esta etiqueta en la URL que les indicaste, la verificación debe completarse.

Comprobaciones útiles:

  • Asegúrate de subir este archivo a la ruta exacta que indicaste en Admitad (ej: /val-de-loire/verificacion-admitad.html).
  • La URL debe devolver HTTP 200 y no 403/401/404.
  • Si usas CDN/Cloudflare, limpia caché o desactiva las reglas que bloqueen bots.

Comandos de prueba (ejecuta en tu terminal):

# Ver cabecera HTTP
curl -I https://alayag82.kesug.com/val-de-loire/verificacion-admitad.html

# Verificar que la meta-tag está en la página (muestra fragmento)
curl -s https://alayag82.kesug.com/val-de-loire/verificacion-admitad.html | grep -i 'verify-admitad' || echo 'meta no encontrada'
